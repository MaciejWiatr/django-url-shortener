# Django URL Shortener 🐍


Simple yet powerful URL shortener written in Python Django

### Features:
  - Url shortening 
  - Generating qr codes 
  - Slick bootstrap design 🔥

### How to run:
  - Clone repo
  - Redirect to cloned directory
  - Run:
```cmd
pip install requirements.txt
```
or
```cmd
pipenv install
```
  - Run:
```cmd
manage.py runserver
```
  - That's it, you're ready to go! 😍
